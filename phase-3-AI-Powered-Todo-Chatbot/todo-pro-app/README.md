# Todo Pro: Manifestation Dynamics (Phase 2)

**Developed by Syed Mujtaba Abbas**

A radical, high-performance manifestation workspace built with **spec-driven development** and the **Apex Protocol** aesthetic.

## Architecture

**Monorepo Structure:**
- `backend/` - FastAPI (Python) REST API with JWT authentication
- `frontend/` - Next.js 16 (TypeScript) with "Apex Protocol" high-fidelity UI
- `specs/` - Feature specifications and design documents
- `.specify/` - Spec-Kit Plus configuration and templates

## Apex Protocol 2.0 (High Fidelity)

This project has evolved into a surgically sharp, polygonal environment designed for peak focus.
- **Apex Geometry**: 45-degree clipped polygonal architecture (`clip-path`) across all cards and dialogs.
- **Dual-Shell Layering**: Multi-layered glass depth where objectives float on inner diagnostic shells.
- **Kinetic Diagnostics**: WebGL Lightning hero section, animated scanline sweeps on hover, and live binary pulse data-streams.
- **Terminal Flow**: Unified monospaced terminal typography and bracketed parameter systems.

## Tech Stack

### Backend
- **Framework:** FastAPI 0.104+

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript 5.x (strict mode)
- **Animations:** Framer Motion (High-Fidelity)
- **Interactive:** WebGL Shaders (Lightning Effect)
- **Styling:** Tailwind CSS 4
- **UI Components:** shadcn/ui + Apex Protocol Geometry
- **Notifications:** Sonner

## Setup Guide

### 1. Backend Protocol
```bash
cd backend
uv init
cp .env.example .env # Configure your DATABASE_URL
alembic upgrade head
fastapi dev src/main.py
```

### 2. Frontend Interface
```bash
cd frontend
pnpm install
cp .env.local.example .env.local # Configure API URL
pnpm dev
```
